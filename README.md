# 🏧 ATM System

A console-based ATM application built in **C++** as part of Dr. Mohamed Abu Hadhood's C++ roadmap.  
Clients can log in using their account number and PIN, then perform transactions directly from the terminal.

---

## 📋 Features

| Feature | Description |
|---|---|
| Quick Withdraw | Choose a preset amount (20, 50, 100, 200, 400, 600, 800, 1000) |
| Normal Withdraw | Enter any custom amount — must be a **multiple of 5** |
| Deposit | Add funds to your account — must be a **multiple of 5** |
| Check Balance | View your current account balance |
| Logout | Securely log out and return to the login screen |

---

## 🔒 Login

- The client logs in with their **Account Number** and **PIN Code**
- Credentials are validated against `BankClients.txt`
- On failed login, an error message is shown and the client is prompted to try again

---

## 💡 Transaction Rules

- **Withdraw (Quick & Normal):** Amount must not exceed the current balance — the system checks and rejects if it does
- **Normal Withdraw & Deposit:** Amount must be a multiple of 5
- **Quick Withdraw:** Fixed preset amounts only (20 → 1000), with an exit option
- After every successful transaction, the **new balance is displayed immediately**
- All changes are saved directly to `BankClients.txt`

---

## 🗂️ Data Storage

This system shares the same data file as the [Bank Client Management System](https://github.com/farida285/Bank-Client-Management-System).

**`BankClients.txt`** — stores client records:
```
A100#//#2002#//#Farida Mahmoud#//#01222263457#//#797000000.000000
```
Fields: `AccountNumber #//# PinCode #//# Name #//# PhoneNumber #//# Balance`

> `BankClients.txt` must exist and contain at least one client record before running the ATM.  
> It is created and managed by the [Bank Client Management System](https://github.com/farida285/Bank-Client-Management-System).

## 🚀 How to Run

### Requirements
- Any C++ compiler (GCC, MSVC, Clang)
- Windows (uses `system("cls")` and `system("pause")`)

### ⚠️ Before You Run

This system reads client data from `BankClients.txt`, which is created and managed by the **Bank Client Management System**.  
You must run the bank system first and add at least one client before using the ATM.

> 🔗 [Bank Client Management System](https://github.com/farida285/Bank-Client-Management-System) — run this first

### Steps
1. Clone the repository
   ```bash
   git clone https://github.com/farida285/ATM-System
   ```
2. Compile the source file
   ```bash
   g++ ATMSystem.cpp -o ATMSystem
   ```
3. Copy `BankClients.txt` (generated by the bank system) into the same folder
4. Run the ATM
   ```bash
   ./ATMSystem
   ```
5. Login with any client's account number and PIN that you added in the bank system
